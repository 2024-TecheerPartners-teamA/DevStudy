## 섹션 5
## Rendering Elements

Element

- 리액트 앱을 구성하는 가장 작은 블록들이라고 보면 됨.
- 화면에서 보이는 것들을 기술

## const element = <h1> hellp, world</h1> 

- 이게 리액트 엘리먼트 코드 적용한 것임
- 자바스크립트 객체 형태로 존재함
- 불변성을 가지고 있음
- 화면에 변경된 엘리먼트 보여줄려면?
- 기존의 엘리먼트를 수정하는 것이 아니라 새로운 엘리먼트를 만들어서 교체하여 화면을 바꿀 수 있음
